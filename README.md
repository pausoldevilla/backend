# backend - Vintage Market

Aquest és el backend de "soldevilla", una plataforma centrada en la compra i venda d’articles de segina ma històrics com mobles, joies i pintures, vestimenta.

## Tecnologies previstes
- Node.js
- Base de dades NoSQL (MongoDB)

## Funcionalitats principals
- Gestió d’usuaris (registre, login, rols)
- CRUD d’articles (crear, llegir, actualitzar, eliminar)
- Sistema de missatgeria entre compradors i venedors
- Gestió de categories i filtres de productes
